Recommended starting points

  • Pomodoro Session Planner: Plan focus sessions and breaks.
  • Focus Time Calculator: Calculate deep work time after interruptions.
  • Task Priority Score: Score task priority.
  • Eisenhower Matrix Helper: Classify a task by urgency and importance.
  • Time Blocking Planner: Estimate number of time blocks.
  • Weekly Review Score: Score weekly completion.
  • Habit Tracker Progress: Track habit progress.
  • Goal Breakdown Calculator: Break a goal into daily targets.

Workflow

Choose the tool closest to your task, run the default example, replace values with your own, and verify the result before using it in important work.
